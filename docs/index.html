What is a Linter?

Software that checks and cleans your code
  • Syntactic Errors
    • Missing Braces/Parens
    • Undefined Variables/Functions
  • Formatting/Preferences
    • Spacing
    • Semis
    • Vars on Top
  • Stylistic Preferences
    • Object shorthand
    • Template literals
    • Yoda Conditionals
There's More
Linters do so much more
They improve your code!
  • Remove Debugging Code
    • Console, Debugger statements
  • Find Dead Code
    • Unused vars, Unreachable code
  • Fix Runtime Errors!
    • Missing vars
    • Assignment in conditionals
    • Strict Equality
    • Missing constructors
It's not just for JS!
Works for templates too.
  • Formatting
    • Indentation
    • Disable inline styling
  • Security
    • triple-curlies
    • rel noopener
  • Better templates
    • Require alt attributes
    • No hard-coded strings (i18n/a11y)
    • Unused vars

You need Linters!

Collections of best practices
